Methodology & Features

How It Works

Every Monday, the pipeline downloads 5 years of S&P 500 (OHLCV) and VIX data, engineers 10 technical features for the most recent week, and retrains both models on a rolling 2-year window (104 weeks). The model with the lower walk-forward RMSE wins and generates the weekly forecast. This week’s forecast: $7,402 — range $7,233–$7,572 (-0.08%) — the model expects a downward week for the S&P 500.

Input Features (10)

All features are computed using data available at or before Friday’s close of week W. The target (next week’s return) uses the Friday W+1 close. No future data ever leaks into the features.

Feature Source Why it matters
RSI (14) S&P 500 weekly Momentum oscillator — signals overbought (>70) or oversold (<30) conditions.
EMA (12) S&P 500 weekly Exponential MA weighted toward recent prices — reacts faster than SMA to trend shifts.
Price / SMA20 Derived How far price has deviated from its 20-week average — mean-reversion signal.
Trend Spread Derived (SMA20 − SMA50) / SMA50 × 100. Positive = short trend above long trend (bullish); negative = compression or downtrend. Price-level independent.
VIX Level CBOE daily (Fri close) Market fear gauge — elevated VIX precedes volatile weeks and risk-off selling.
VIX Change CBOE daily (derived) Week-over-week % change in VIX. A spike matters more than the absolute level — captures fear acceleration.
Lag Return −1W S&P 500 weekly Previous week's return — short-term momentum signal.
Lag Return −2W S&P 500 weekly Two-week-ago return — momentum continuation signal.
Lag Return −3W S&P 500 weekly Three-week-ago return — baseline for momentum decay pattern.
Realized Vol S&P 500 daily Std dev of 5 daily returns within the week — current volatility regime proxy.

Why XGBoost & Random Forest?

Financial return data is noisy and non-linear — linear regression consistently underperforms. Both tree-based ensembles handle this well without manual feature interaction engineering.

XGBoost
Gradient boosting: each tree corrects the residuals of the previous one. Aggressive learner — excels when the signal is consistent.
Random Forest
Bagging ensemble: many uncorrelated trees vote together. Conservative learner — more robust when data is noisy or regime-shifting.

By running both every week and picking the winner by walk-forward RMSE, the pipeline adapts to whichever model fits the current market regime better.

Walk-Forward Validation & No Data Leakage

  • TimeSeriesSplit (5 folds): in every fold, training data always precedes validation data in time. Standard k-fold would mix future data into training — a critical flaw in time-series forecasting.
  • RobustScaler over StandardScaler: financial returns have extreme outliers (COVID crash, flash crashes). RobustScaler uses median + IQR instead of mean + std, making it resistant to those spikes.
  • Rolling 2-year window: the market of 2020 behaves structurally differently from 2024. Training on all available history would dilute current-regime patterns; 104 weeks keeps the model anchored to recent market dynamics.
  • Scaler fit on train only: the RobustScaler is fitted exclusively on training rows and then applied to the prediction row — never the reverse.
